# 设计说明



## 配色方案

- 主色：#D4C5A0
- 辅助色：#FFFFFF
- 强调色：#2D5016



## 字体

- 标题：Inter、Noto Serif SC
- 正文：font family/Font 1、Inter



## 核心页面

1. 首页 - 介绍网页大致功能及工作内容
2. 登录页 - 用户登录
3. 注册页 - 用户注册
4. 项目管理页 - 添加、管理及修改项目
5. 文件管理页 - 对项目中的文件进行管理（导入导出及新建）
6. 文档详情页 - 包含对文档的实体标注、古文解析及自动分词功能



## 交互说明

- 首页点击登录跳转至登录页面，若无账号可点击“立即注册”进入注册页面
- 项目管理页点击各文件进入各文件的文件管理页面
- 文件管理页中点击各文档进入文档详情页，点击返回即返回至项目管理页
- 文档详情页分别点击三个核心功能按钮，右下部分的内容框会随之显示出相应的功能。点击返回即返回至文件管理页。
