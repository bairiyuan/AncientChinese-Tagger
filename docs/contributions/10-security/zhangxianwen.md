#### 审查的文件与模块

| 文件/模块 | 审查范围 |
| --------- | -------- |
| .github/workflows/security.yml | GitHub Actions 安全工作流 |
| backend/.env.example | 环境变量模板安全配置 |

#### 发现的主要问题

| 问题类型 | 风险等级 | 说明 |
| -------- | -------- | ---- |
| 缺乏自动化安全扫描 | 高 | 无法自动检测提交代码中的密钥泄露风险 |
| 启动文档凭据泄露 | 高 | local-startup.md 中包含硬编码的数据库密码 |
| 数据访问权限限制过严 | 中 | 导致用户无法正常查看系统内的公共/存量项目 |
| UI 布局溢出重叠 | 低 | AI 回复内容过长导致界面元素重叠，影响信息展示 |

#### 我修复的问题

| 问题类型 | 修复文件 | 修复方式 |
| -------- | -------- | -------- |
| 缺乏自动化安全扫描 | security.yml | 集成 Gitleaks Action，实现每次 Push 自动进行密钥扫描 |
| 启动文档凭据泄露 | local-startup.md | 移除所有硬编码凭据，改为引导用户使用本地 .env 文件 |
| 数据访问权限限制过严 | projects_service.py | 优化 list_projects 逻辑，允许查看所有项目，提升系统可用性 |
| UI 布局溢出重叠 | EditorView.vue | 优化 AI 问答组件 CSS，增加 max-height 与滚动支持 |
