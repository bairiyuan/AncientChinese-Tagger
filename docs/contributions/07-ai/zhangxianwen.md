# 个人贡献文档 - 张贤文 (zhangxianwen)

## 核心贡献概览
在本项目中，我主要负责了 **AI 赋能模块的深度集成**、**项目管理功能的完整联调**以及**编辑器交互体验的系统性优化**。通过接入 DeepSeek 大语言模型，实现了古文自动标注、智能解析与分词，显著提升了系统的智能化水平。

---

## 详细贡献内容

### 1. AI 智能化增强 (AI Integration)
- **DeepSeek 模型接入**：在后端封装了 `llm_service`，实现了与 DeepSeek API 的深度集成。
- **古文智能解析**：设计并优化了针对古汉语的 System Prompt，支持对文本进行精准的断句、语法分析及现代文翻译。
- **AI 自动标注系统**：开发了高召回率的实体识别逻辑，能够自动从古文中提取人名、地名、时间及核心概念，并支持一键批量入库。
- **双引擎分词机制**：实现了“DeepSeek 优先 + 本地 Jieba 降级”的分词方案。当 AI 响应超时时，系统会自动切换至本地 Jieba 引擎，确保服务的高可用性。
- **智能对话助手**：在编辑器中集成了上下文感知的 AI 助手，支持针对当前文本进行深度问答。

### 2. 项目管理模块联调 (Project Management)
- **实时数据统计**：开发了后端统计接口，实现了文档总数、标注总数及实体类型分布（饼图/环形图）的真实数据展示。
- **持久化重命名**：完成了项目名称与描述编辑功能的前后端联调，支持数据的实时同步更新。
- **个人中心增强**：在登录与个人信息模块增加了注册时间字段，提升了用户信息的完整度。

### 3. 编辑器交互与 UI 优化 (UI/UX Optimization)
- **左右等高布局**：重构了 `EditorView` 的 CSS 架构，实现了左侧正文区与右侧功能区的动态等高对齐，极大提升了视觉平衡感。
- **侧边栏整体滚动**：引入了“滚轮式”侧边栏交互，解决了功能面板过多导致页面拉伸的问题。
- **Markdown 渲染适配**：集成了 `markdown-it` 引擎，完美解决了 AI 回复中黑体加粗、列表等格式的显示问题。
- **标注排序逻辑**：实现了已有标注按文档位置（`start_pos`）的自动排序，方便用户有序对照。
- **自动滚屏体验**：为 AI 对话框增加了消息自动置底逻辑，确保最新回复即时可见。

## 技术栈应用
- **后端**: Python (FastAPI), SQLAlchemy, DeepSeek API, Jieba
- **前端**: Vue 3 (Composition API), TypeScript, Markdown-it, CSS Grid/Flexbox
- **版本控制**: Git (GitHub 分支管理与规范化提交)

## 总结
通过本次开发，我成功将古文标注工具从一个基础的 CRUD 应用转型为具备 AI 辅助分析能力的智能化平台。在保证系统稳定性的前提下，大幅优化了用户在处理长篇古籍时的操作效率与阅读体验。
